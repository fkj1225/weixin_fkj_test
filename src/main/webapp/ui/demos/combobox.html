可输入

禁止输入

调用方法：$('select').combobox()；可输入，带自动完成功能 

		//最简单用法
		$('#select1').combobox();
		//禁止输入
		$('#select2').combobox({
			readonly: true
		});

2、两个回调事件，select,change的用法

两个回调都返回两个参数，分别是事件对象和当前选中的option对象 

		$('#select3').combobox({
			select: function(event,option){
				$.alert('select回调，选择的项value=' + option.item.value + '; text=' + option.item.text);
			},
			change: function(event,option){
				$.alert('change回调，选择的项value=' + option.item.value + '; text=' + option.item.text);
			}
		})
		//支持原生的事件绑定
		.change(function(event){
			$.alert('自己绑定的change事件，选择的是value=' + this.value + '; text=' + this[this.selectedIndex].text);
		});

		//注：不建议绑定原生的change事件，
		//之所以支持是为了兼容之前已经写好的应用，
		//建议后续的使用过程使用回调

3、方法select：设置当前选中项

通过value来设置：

通过text来设置：

设置当前选中项，可通过value、text来设置 

		//初始化
		$('#select4,#select5').combobox();

		//设置选中 value="2" 的项为当前选中项
		$('#select4').combobox('select', {value: '2'});

		//设置选中 value="东莞" 的项为当前选中项
		$('#select5').combobox('select', {text: '东莞'});
