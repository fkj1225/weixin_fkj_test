1、拿页面直出的json数据

设置data

data为json格式的数据：[{},{},...]; 数组里面每个项的格式为{ id: 1, text: '树节点名称', child: [], show: true}，child表示该节点有字节点，show是显示的意思，默认不显示 

		$('#tree1').tree({
			data: treeData[1]
		});

2、设置惰性加载、带checkbox和有点击节点后的回调

惰性加载设置lazyload: true 

		$('#tree2').tree({
			data: treeData[0],
			hasCheckbox: true,
			lazyload: true,
			click: function(e, nodeData){
				alert('节点id='+nodeData.id + '; 节点text=' + nodeData.text);
			}
		});
