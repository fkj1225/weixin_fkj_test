1、复制点击节点的内容

点击复制链接

设置path和copy，copy的方法必须要返回；注意：这个插件是flash插件，本地无法调试 

			$('#copyLink').zclip({ 
			        path: '../zclip/ZeroClipboard.swf', 
			        copy: function(){//复制内容 
			            return $(this).attr('href'); 
			        }, 
			        afterCopy: function(){//复制成功 
			            alert('复制成功') 
			        } 
			    });
