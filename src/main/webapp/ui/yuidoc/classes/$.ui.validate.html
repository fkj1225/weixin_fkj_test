Defined in js\validate.js:44

匹配规则配置项，默认配置有email,mobile,tel,qq,url,noBlank，可自定义覆盖默认

配置项为一个数组，[正则|函数，验证错误的提示信息，是否可以为空（可选，用于一些需要验证的但不是必填项，true表示可以为空）]

Example:

regexp: { idcard: [/^\d{17}[\dxX]$/, '请输入正确的身份证号码'] }
