1、最简单用法

只需指定一个target即可，默认下拉的层是位于点击目标的下方，且相对于点击目标左右居中 

		$('#btn1').downlayer({
			target: '#m1'
		});

2、目标点击后带状态

在配置项设置tgClass，点击后为目标增加一个样式 

		$('#btn2').downlayer({
			target: '#m1', 
			tgClass: 'clicked'
		});

3、设置偏移量

在配置项设置margin，这个例子是相对左偏移10px，相对上偏移10px，值可以为负数 

		$('#btn3').downlayer({
			target: '#m1', 
			margin: { l: 10, t: 10}
		});

4、设置居右

在配置项设置position为“r”即可，上下左右分别为：t,b,l,r 

		$('#btn4').downlayer({
			target: '#m1', 
			position: 'r'
		});

5、回调

有两个回调，分别是：显示下拉层之前触发的beforeShow、隐藏下拉层之前触发的beforeHide。 

		$('#btn5').downlayer({
			target: '#m1',
			beforeShow: function(event){
				alert('显示下拉层之前要干的活写在这里');
			},
			beforeHide: function(event){
				alert('隐藏下拉层之前要干的活写在这里');
			}
		});
