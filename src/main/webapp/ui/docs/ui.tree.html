Namespace: ui.tree

ui.tree

树插件
1、支持惰性加载
2、可设定默认显示具体节点下的子节点
3、可设置checkbox
4、节点数据可以页面直出，也可以ajax获取，但都必须是json格式
Version:
  • 1.1
Author:
  • derrickliu
Source:
  • tree.js, line 1

Example

$(selector).tree({
	hasCheckbox: false,
	url: 'http://xx.xx', //(或者 data: [],如果两个参数都有，url优先)
	idAttribute: 'treeId',
	//点击树节点的回调
	click: function(event,nodeData){}
});
